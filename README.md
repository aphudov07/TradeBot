Общая логика:

•	Торгуется актив: XAUUSD (золото к доллару)
•	Таймфрейм: M1 (1 минута), тренд берётся с M15
•	Работа 24/7 с трейлинг-стопом и автоматическим входом/выходом
•	Платформа: MetaTrader 5 (через MetaTrader5 Python API)
•	Режим: полностью автономный + управление через Telegram
________________________________________
Стратегия (основа для входа в сделку):
1.	EMA20 > EMA50 (или наоборот для шортов)
2.	Высокий объём (tick_volume > скользящее среднее)
3.	Бычья или медвежья свеча
4.	ATR > 0 (активность рынка)
5.	Фильтрация по M15 — направление тренда
________________________________________
Параметры сделок:
•	Объём: 0.1 лота
•	SL/TP: считаются от текущей цены + ATR × множитель
•	Соотношение риск/прибыль: 1 к 2
•	Один вход на свечу, защита от дублирования сделок
•	Трейлинг-стоп:
o	Активация при +0.5$
o	Шаг: 0.25$ каждые 1 секунду
________________________________________
Логирование:
•	Все сделки записываются в trade_log.csv с результатом
•	Файл содержит: timestamp, direction, price, sl, tp, lot, result
________________________________________
Визуализация:
•	При открытии сделки бот сохраняет график (цена + EMA + TP/SL)
•	Скриншот отправляется в Telegram
________________________________________
Telegram-команды:
Команда	Описание
/pause	Приостановить торговлю
/resume	Возобновить торговлю
/closeall	Закрыть все активные сделки
/status	Баланс и открытые позиции
/stats	Статистика: прибыль, winrate, макс. прибыль/убыток
/last	Последняя завершённая сделка
/profit	Текущий плавающий PnL
/report	Визуальный отчёт: графики прибыли, побед, активности
/help	Все команды
